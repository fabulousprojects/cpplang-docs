About

The letters cpps stand for c plus plus simplified. After releasing the first prototype on 19th of february 2024 the cpps language is yet to receive any further updates. With only the feature for printing a peice of text.

Setup

The setup is fairly simple with only 3 steps.


Step 1: Download the installer from our official github page


Step 2: Make a file with the extension cpps and with the syntax specified below


Step 3: Open the compiler and type in only the name of the file and not the file extension and press enter to run

Syntax


Printing is fairly simple with the syntax being print "some text"


But as a easter egg running print default gives a suprise answer
